# 基于Vite6的React19项目全家桶

## 项目集成说明

1. 支持React 19
2. 支持Sass/Scss/Less/Stylus
3. 已安装react-router-dom
4. 已安装ReduxToolkit、react-redux
5. 已安装axios
6. 已安装mockjs
7. 已安装antd 5.x
